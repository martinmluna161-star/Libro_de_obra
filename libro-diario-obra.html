Instrucciones de Deploy

  1. Abrí script.google.com → Nuevo proyecto → pegá el contenido de Code.gs
  2. Menú Implementar → Nueva implementación
  3. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona
  4. Autorizá los permisos de Google Sheets y Drive
  5. Copiá la URL generada y pegala arriba en "URL del Apps Script Web App"
  6. Subí los tres archivos HTML a tu repositorio de GitHub
  7. Activá GitHub Pages en Settings → Pages → rama main
  8. El link del reporte para el cliente quedará del tipo:
    https://TU-USUARIO.github.io/REPO/reporte-cliente.html?proyecto=NombreProyecto
